Full crowns data

## Usage

``` r

```

## Format

A tibble with 131,656 observations on the following 23 variables:

- `site` : site name

- `id`: Crown id

- `date`: date of the rgb survey

- `family`: family

- `genus`: genus

- `species`: species

- `type`: type

- `metric`: metric

- `band`: band

- `value`: value7
