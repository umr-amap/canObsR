# Visualisation shiny app

Shiny app to do the visualize the crowns data (labels and rgb indices).

## Usage

``` r
shiny_visualisation(data_labeling)
```

## Arguments

- data_labeling:

  tbl_df. Labeling file resulting from the
  [`create_labelingFile`](https://umr-amap.github.io/canObsR/reference/create_labelingFile.md)
  and modify from the
  [`shiny_labels`](https://umr-amap.github.io/canObsR/reference/shiny_labels.md)

## Examples

``` r
library(canObsR)

# shiny_visualisation() # Run it
```
