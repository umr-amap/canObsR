# RGB indices data

RGB indices data

## Usage

``` r
data("rgb_data")
```

## Format

A data.frame with 131,656 observations on the following 10 variables:

- `site`: site name

- `id`: Crown id

- `date`: date of the rgb survey

- `family`: family

- `genus`: genus

- `species`: species

- `type`: type

- `metric`: metric

- `band`: band

- `value`: value

## Examples

``` r
data(rgb_data)
```
